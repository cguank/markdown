# 找链表倒数的结点
## 思路
可以使用 **快慢双指针**
倒数第n个节点的位置为```L-n+1```，前一个节点为```L-n```
1. 设置dummy节点，位置为0
2. 快指针先走n步
3. 快指针走到尾节点需要L-n步
4. 慢指针也走L-n步，位置恰好是倒数第n个节点的前一个节点
### 证明
假设链表长L，要找倒数第n个结点，快指针fast，慢指针slow
走到尾需要```L-n步```
| 结点 | fast走了n步后的位置 | fast走到尾的位置 |
| ---- | ---- | ---- |
| fast | n | L |
| slow | 0 | L-n |
**此时slow恰好是倒数节点的前一个节点**
